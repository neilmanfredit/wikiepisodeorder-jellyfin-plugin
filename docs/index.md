# Wikipedia Episode Order — Jellyfin Plugin Repository

Add this repository to Jellyfin:

**Repository URL:** `https://neilmanfredit.github.io/wikiepisodeorder-jellyfin-plugin/manifest.json`

Then install "Wikipedia Episode Order" from Dashboard → Plugins → Catalogue.

See the [README](https://github.com/neilmanfredit/wikiepisodeorder-jellyfin-plugin) for full installation and usage instructions.
